UNDERSTANDING THE CRYPTOGRAPHY

The Modular Mathematics

Letters (A-Z) are mapped to numeric values from 0 to 25. The core algorithm executes mathematical offsets using Modular Arithmetic:

Encryption: Ci = (Pi + Ki) mod 26 Decryption: Pi = (Ci - Ki + 26) mod 26

Where P is the plaintext, K is the key, and C is the ciphertext.

Polyalphabetic Substitution

Unlike Caesar's cipher (monoalphabetic), the Vigenère cipher uses a keyword to alternate shifts. Because the key repeats, the same plaintext letter can result in different ciphertext letters.

For example, with key "KEY", the word "LLL" encrypts to "VJD". This flattens letter frequency peaks, making basic frequency analysis attacks ineffective.

Historical Cryptanalysis

Once called "le chiffre indéchiffrable" (the indecipherable cipher), it was eventually broken by Friedrich Kasiski in 1863. Kasiski found that repeating text patterns reveal the keyword length.

By finding factors of distances between repeating ciphertext sequences, attackers determine the key length and solve it as multiple interleaved Caesar ciphers.
